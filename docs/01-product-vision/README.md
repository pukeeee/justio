# Розділ 1: Бачення Продукту

**Версія:** 1.0  
**Дата:** 26 грудня 2025  

## 1.1. Місія та Концепція

Наша місія — надати малому та середньому бізнесу (МСБ) в Україні доступний, потужний та інтуїтивно зрозумілий CRM-інструмент, що централізує управління клієнтами та замовленнями. Продукт розробляється як веб-застосунок, оптимізований для мобільних пристроїв (mobile-first) та здатний працювати офлайн як PWA (Progressive Web App).

Ключова мета — оптимізувати щоденні робочі процеси підприємців: відстеження лідів, управління продажами, ведення клієнтської бази та інтеграція з популярними локальними сервісами (наприклад, «Нова Пошта»).

## 1.2. Стратегічний Розвиток: від CRM до B2B2B Платформи

Бачення продукту виходить за межі простої CRM. Початкова мета — створити найкращий у своєму класі B2B CRM для окремих компаній. У майбутньому платформа еволюціонує у **B2B2B мережу**, де постачальники зможуть приєднуватися до системи, ділитися каталогами своїх товарів та отримувати замовлення від бізнес-клієнтів.

Ця екосистема дозволить підприємцю не лише керувати власними клієнтами, але й переглядати асортимент постачальників та розміщувати замовлення на поповнення запасів — все в межах єдиної системи.

## 1.3. Цільова аудиторія

| Сегмент | Опис | Розмір команди | Ключові проблеми (Pain Points) |
|---|---|---|---|
| **Мікробізнес** | Торгові точки, СТО, салони краси, кафе | 1-5 осіб | Ведення обліку в Excel/блокнотах, втрата інформації про клієнтів, відсутність історії взаємодій. |
| **Малий бізнес** | Невеликі магазини, клініки, B2B-компанії | 5-20 осіб | Хаотичні процеси, втрата лідів, відсутність базової аналітики, неефективна комунікація в команді. |
| **Середній бізнес** | Роздрібні мережі, дистриб'ютори, сервісні компанії | 20-100 осіб | Потреба в автоматизації рутинних завдань, кастомних інтеграціях, гнучкому налаштуванні прав доступу. |

## 1.4. Ключові диференціатори

1.  **Offline-first** — критично важлива функціональність для України, що дозволяє працювати навіть за відсутності стабільного інтернет-з'єднання.
2.  **Mobile-first** — понад 70% операцій спроєктовано для виконання зі смартфонів, забезпечуючи мобільність для менеджерів та власників.
3.  **Локальні інтеграції** — готова інтеграція з «Нова Пошта» та іншими українськими сервісами "з коробки".
4.  **Українська локалізація** — не просто переклад, а глибока адаптація бізнес-процесів під реалії українського ринку.
5.  **Доступність** — наявність безкоштовного тарифного плану для старту та конкурентні ціни для зростаючих компаній.
6.  **Масштабованість та модульність** — архітектура, що дозволяє швидко розвивати продукт. На старті використовується підхід **"модульного моноліту"**, що дає змогу швидко ітерувати. В майбутньому це дозволить легко додавати нові модулі (склад, аналітика), а також створити **відкритий API** та **маркетплейс додатків** для розширень від сторонніх розробників.
7.  **Вбудована монетизація** — система з першого дня проєктується з урахуванням моделі підписки, що дозволяє гнучко керувати функціоналом для різних тарифних планів.
